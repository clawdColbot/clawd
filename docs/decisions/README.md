# Decision Records

Este directorio contiene registros de decisiones arquitectónicas importantes (ADRs - Architecture Decision Records).

## Formato

```markdown
# ADR-NNN: Título de la decisión

**Fecha:** YYYY-MM-DD  
**Estado:** Propuesto / Aceptado / Rechazado / Obsoleto  
**Decisor:** ClawdColombia

## Contexto

¿Qué problema estamos resolviendo? ¿Qué restricciones existen?

## Opciones Consideradas

### Opción 1: [Nombre]
- ✅ Pros
- ❌ Contras

### Opción 2: [Nombre]
- ✅ Pros
- ❌ Contras

## Decisión

Elegimos **Opción X** porque...

## Consecuencias

### Positivas
- Beneficio 1
- Beneficio 2

### Negativas
- Costo 1
- Costo 2

## Notas

Información adicional, links, referencias.
```
